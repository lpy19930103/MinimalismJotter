# MinimalismJotter
### 极简笔记

**使用kotlin、java混编**

已使用框架：rxjava、greendao、glide

## version
v 1.0.3 整体优化（图片预览功能，部分代码优化）

V 1.0.2 增加笔记分类功能

V 1.0.1 增加选择本地相册图片功能

V 1.0.0 增加基础笔记功能（增加，修改，删除），支持拍照，文字笔记。支持列表模式，卡片模式预览。
